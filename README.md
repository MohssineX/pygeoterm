# pygeoterm

![Python](https://img.shields.io/badge/python-3-blue)
![License](https://img.shields.io/badge/license-GPLv3-blue)

A lightweight terminal application for calculating the areas of common geometric shapes directly from the command line.

## Features

* Calculates the area of 7 common geometric shapes
* Built-in help menu (`--help`)
* Custom measurement unit support (cm, m, km, etc.)
* Pure Python implementation with no external dependencies

## Supported Shapes

* Square
* Rectangle
* Triangle
* Circle
* Parallelogram
* Trapezoid
* Rhombus

## Supported Platforms

- Windows
- Linux
- macOS

## Requirements

* Python 3.6 or later
* Terminal with ANSI escape sequence support

## Installation

### Clone the repository :

```bash
git clone https://github.com/MohssineX/pygeoterm.git
cd pygeoterm
```

## Usage

Run the program :

```bash
python pygeoterm.py
```

If your system uses `python3` :

```bash
python3 pygeoterm.py
```
